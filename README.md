# **🗺️ Computer Science 1: The Java Bridge**

**Full Year Syllabus & Pacing Guide**

## **🍎 Course Overview**

Welcome to CS1\! Whether you've been coding since middle school or this is your very first time typing a line of code, this course is designed for you.

We will use the **Java** programming language and the **BlueJ** environment to build games, solve puzzles, and understand how software powers the world.

🎯 **The Ultimate Goal:** This course is the official bridge to **AP Computer Science A**. By May, you will be writing college-level code.

## **🎒 The "Mixed Classroom" Approach**

Every lab in this textbook has two levels:

* 🍔 **The Core:** The main assignment. Everyone must complete this to master the standard (TEKS).  
* 🌶️ **The Spicy Extension:** For our magnet students and those craving a challenge. These push your logic skills to the AP level.

## **📅 Quarter 1: The Visual World of Objects (Weeks 1-9)**

*The goal of Q1 is to build confidence. We use BlueJ's visual interface to interact with pre-written code before we write our own.*

* **Week 1-2: Welcome to BlueJ** 📦  
  * Objects, Classes, and the BlueJ grid.  
  * *Lab:* The Shapes Canvas (Making a house out of code blocks).  
* **Week 3-4: Methods & Parameters** 📥  
  * Passing data to objects (e.g., moveRight(50)).  
  * Data types preview: int, String, boolean.  
* **Week 5-6: Reading the Source Code** 🕵️‍♂️  
  * Opening the BlueJ editor. Looking at the Java Syntax Cheat Sheet.  
  * 🏰 Curly braces, 🛑 semicolons, and 📝 comments.  
* **Week 7-9: Writing Basic Methods** ⚙️  
  * Modifying existing classes.  
  * *Q1 Project:* "The Animated Scene" (Using method calls to create a moving picture).

## **📅 Quarter 2: Data, Math, and Decisions (Weeks 10-18)**

*Now we take the training wheels off. We start writing our own logic and making the computer do the math.*

* **Week 10-12: Primitives & Math** 🔢  
  * Declaring variables (int, double).  
  * Arithmetic operators (+, \-, \*, /, %).  
  * 🌶️ *Spicy:* The Modulo operator (%) for odd/even logic.  
* **Week 13-15: Boolean Logic & Conditionals** 🚦  
  * if, else if, and else statements.  
  * Relational operators (==, \>, \<).  
* **Week 16-18: Complex Decisions** 🔀  
  * Compound logic: AND (&&), OR (||), NOT (\!).  
  * *Q2 Project:* "Text-Based Adventure Game" (Choose your own adventure using nested if statements and user input).

## **📅 Quarter 3: Blueprinting \- Writing Classes (Weeks 19-27)**

*This is the AP CSA transition quarter. We stop using pre-built BlueJ shapes and start writing our own classes from absolute scratch.*

* **Week 19-21: Anatomy of a Class** 🏗️  
  * Instance variables (State) 📂.  
  * Constructors (The setup routine) 🐣.  
* **Week 22-24: Methods & Return Types** 🎁  
  * Void methods vs. Return methods.  
  * The this keyword.  
  * 🌶️ *Spicy:* Writing Getter/Setter methods perfectly.  
* **Week 25-27: The while Loop** 🔄  
  * Repeating code until a condition is met.  
  * Avoiding the dreaded 💥 Infinite Loop\!  
  * *Q3 Project:* "The Tamagotchi / Virtual Pet" (Students design a class from scratch with health, hunger, and happiness stats that loop).

## **📅 Quarter 4: Algorithms & The AP Prep (Weeks 28-36)**

*The final stretch. We introduce the structures needed for AP CSA next year.*

* **Week 28-30: The for Loop** 🎡  
  * Counting loops and standard algorithms.  
  * 🌶️ *Spicy:* Nested loops (loops inside loops) to draw ASCII art.  
* **Week 31-33: 1D Arrays** 🚂  
  * Storing multiple values in a single variable.  
  * Array indexing (Starting at 0\!).  
  * Iterating through arrays with loops.  
* **Week 34-36: Strings & Final Projects** 🧵  
  * Introduction to String methods (length, substring, equals). *Crucial for AP\!*  
  * *Q4 Final Project:* "The Arcade" (A culminating BlueJ project combining custom classes, loops, arrays, and logic).
